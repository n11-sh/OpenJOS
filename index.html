OpenJOS

This project does not guarantee anything at 100 percent. It might be unstable or behave strangely because it’s being built by a single teenager experimenting with system design, not a company or team of experts.
OpenJOS is an experimental Linux distribution where the core tools and system services are built in Java. The kernel remains Linux-based in C, but everything else aims to create a modern, minimalist, and secure environment. It’s a playground for learning, exploring, and seeing what’s possible when we rethink the way a Linux system works.
OpenJOS is an independent project and is not affiliated with Oracle or the official Java platform.
